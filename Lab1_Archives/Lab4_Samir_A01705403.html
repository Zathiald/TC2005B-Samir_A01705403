¿Qué diferencias y semejanzas hay entre Java y JavaScript?

Semejanzas:

  • Ambos son lenguajes de programación orientados a objetos.
  • Ambos pueden manejar eventos y soportar la programación concurrente.

Diferencias:

  • Tipo de Lenguaje:Java es un lenguaje de programación de propósito general principalmente orientado a objetos, compilado y estáticamente tipado12. JavaScript es un lenguaje de programación interpretado, dinámicamente tipado y se utiliza principalmente para la programación del lado del cliente en páginas web.
  • Plataforma:Java se ejecuta en la Plataforma Java, lo que permite que las aplicaciones Java sean altamente portátiles y se puedan ejecutar en diferentes sistemas operativos1. JavaScript se utiliza principalmente en navegadores web para agregar interactividad a las páginas web.

¿Qué métodos tiene el objeto Date? (Menciona al menos 5*)

El metodo Date es una forma de obtener la fecha exacta en un dispositivo, algunos metodos que tiene son:

  1. getMonth(): Devuelve el número de mes del 0 (enero) al 11 (diciembre)
  2. getDate(): Devuelve el día del mes, del 1 al 31
  3. getDay(): Devuelve el día de la semana del 0 (domingo) al 6 (sábado)
  4. getHours(): Devuelve la hora actual, desde 0 a 23 (formato 24 horas)
  5. getMinutes(): Retorna los minutos

¿Qué métodos tienen los arreglos? (Menciona al menos 5*)

Los arreglos son formas de almacenar informacion de dieferentes datos en una lista, estos tienen metodos como son:

  1. push(): Agrega uno o más elementos al final del arreglo y devuelve la nueva longitud del arreglo
  2. pop(): Elimina el último elemento de un arreglo y lo devuelve
  3. shift(): Elimina el primer elemento de un arreglo y lo devuelve
  4. unshift(): Agrega uno o más elementos al inicio del arreglo y devuelve la nueva longitud del arreglo
  5. splice(): Cambia el contenido de un arreglo eliminando, reemplazando o agregando elementos10.

¿Cómo se declara una variable con alcance local dentro de una función?

Para declarar una variable con alcance local dentro de una función en JavaScript, puedes usar la palabra clave let dentro del cuerpo de la función.

¿Qué implicaciones tiene utilizar variables globales dentro de funciones?

Las variables globales en JavaScript son accesibles desde cualquier parte del código, incluso dentro de las funciones. Sin embargo, su uso puede tener varias implicaciones:

  • Dependencias ocultas: Las variables globales crean dependencias ocultas en el código, lo que puede dificultar la comprensión y el mantenimiento del código.
  • Riesgo de conflictos: Si diferentes partes del código intentan modificar la misma variable global, puede dar lugar a comportamientos inesperados.
  • Dificultad para el testing: Las variables globales pueden hacer que las pruebas sean más difíciles de escribir, ya que las pruebas pueden necesitar configurar el estado global antes de ejecutarse. Por estas razones, generalmente se recomienda limitar el uso de variables globales en JavaScript
